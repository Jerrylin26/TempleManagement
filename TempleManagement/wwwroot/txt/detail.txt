需求介紹:

	1. 信徒資本資料: BasicInfo
		
		# 已建置
		DB [BasicInfo]
			[Primary Key : MID] 個人編號 *虛擬 Primary key
			姓名 Name
			性別 Sex
			生肖 Zodiac
			年齡 Age
			室話 Home_num
			行動電話 Phone
			職業 Job
			類別 Character_type
			農曆生日 Lunar_birthday
			國曆生日 Birthday
			身分證字號 Identical_num
			戶籍地址 Household_address
			通訊地址 Current_address
			郵遞區號 Postal_code_cur
			郵遞區號(通訊) Postal_code_household
			備註 Note

		DB [HouseholdMember]
			[Primary Key : HouseholdID] => 一個人可以有2筆以上資料。意謂著換戶號 => 對應 Household HID
			MemberID  => 對應BasicInfo MID
			編號 Member_no
			戶號 House_ID
			戶長與否 Is_head
			開始日期 Start_date
			結束日期 End_date

		DB [Household] => 用來記錄是否停用 (其實可以與 HouseholdMember 合併)
			[Primary Key : HID] => 流水號 
			是否持續使用該戶號 Is_using
			* CRUD

	2. 捐獻類別管理:
		
		DB[DonateType]
			[Primary Key : id] 
			name 名稱
			price 價格
			note 備註
			modifydate 修改日期
		
		DB [Donation_Household] 一戶
		
			[Primary Key : DH_ID]
			Is_Dipper 是否點燈
			HouseID 戶號 => 對應戶

			(燈別) 3組
			大斗 Dipper_big 4800 (一戶)
			小斗 Dipper_small 500 *同上
			安太歲 Is_Taisui (一戶) 500
			平安燈 Is_PeaceLight (一戶) 200  (前提為安斗) 燈都要安斗
			備註 Note


		DB [Donation_Individual] 個人
		* 前提為安斗，才能光明燈

			[Primary Key : DI_ID]
			MID  個人編號 => 對應MemberID
			=> 用bit
			光明燈(一) 600  BlessingLight600
			光明燈(二) 700  BlessingLight700
			光明燈(三) 800  BlessingLight800
			光明燈(四) 1000 BlessingLight1000	
			備註 Note
		
			* CRUD

	3.  信徒捐獻作業:
		DB [Donation_Operation] 捐獻作業

			[Primary Key : DonationID]  捐獻單號
			MID => 對應BasicInfo 的 MemberID
			捐獻類別 Donation_Type (以戶為單位的只加入戶長帳下)
			日期 Date
			金額 Price
			備註 Note

			* CRUD

	4. 戶家郵遞標籤:
		姓名、電話、性別、信徒編號、類別  * R
		* 報表類型 : 標籤1" 、 1.5" 、 A4
		* 列印方向: 橫印、直印
		* 標籤張數

		DB [MailInfo] 戶家郵遞資訊

			[Primary Key : MailInfo_ID] 郵遞資訊ID
			報表類型 PrintType
			列印方向 PrintDirection
			列印張數 Amount

	5. 信徒清冊:
		姓名、電話、性別、信徒編號、類別  * R
		內容: 備註:年齡、生肖

	6. 信徒點燈疏文:
		燈別類型 & (信徒疏文 | 安斗謝斗)

		DB [PrayerDocument] 疏文
			
			[Primary Key : PD_ID] 疏文ID
			路徑 Path

	7. 信徒捐獻查詢:
		(燈別 || 信徒編號)
		內容: 日期、金額

	8. 信徒增歲作業:
		所有人年齡 +- = 1

	9. 戶長變更作業:
		更換戶長

	10. 帳密作業管理:
		新增刪除修改帳密
		
		DB [Admin] 管理員

		[Primary Key : AdminID]
		帳號 Account
		密碼 Password
